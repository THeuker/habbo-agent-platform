# Nitro Bundle

`nitro/` contains the hotel frontend emulation stack.

## What it provides

- `nitro-react/` client UI
- `nitro-assets/` runtime assets
- `nitro-swf/` SWF/runtime compatibility assets
- `nitro-converter/` conversion tooling

## Related service

`nitro-imager/` is a companion image rendering service used by frontend/profile image flows.
